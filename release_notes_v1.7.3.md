# O que há de novo na v1.7.3

- **Recuperação Automática de Erros do Updater**: Nesta atualização, corrigimos a forma como o app lidava com panes ou bloqueios de permissão durante uma atualização automática via GitHub.
- Agora, caso ocorra alguma falha na "Auto Atualização", em vez de travar o app, ele exibirá um alerta visual pop-up na tela com o log detalhado do motivo da falha. 
- E o melhor: logo após mostrar o erro, ele abrirá de forma automática a página oficial de Release no Github no seu navegador padrão para que você possa baixar o instalador de forma manual, sem interromper seu fluxo de trabalho.
